Error with the underlying integration of technologies.

If you see this error while interacting with the interface :

There is a problem of communication between your local components.

The interface cannot draw results from your database because of inner database issues (latency).

Try to relaunch your local MongoDB.

If you entered your own URL :

The ID your provided in the URL is wrong : no notes exists for this id.

Do not hesitate to check the API in charge of the patient notes and your local MongoDB.
